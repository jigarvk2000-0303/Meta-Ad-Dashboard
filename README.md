Project: Meta Advertising Performance Dashboard
Tools: Microsoft Power BI, Microsoft Excel

Developed an interactive advertising performance dashboard in Microsoft Power BI to analyze campaign metrics across Meta Platforms channels including Facebook and Instagram.

Tracked key marketing KPIs such as impressions, clicks, CTR, engagement rate, conversions, purchases, and advertising budget.

Created visualizations for demographic analysis including impressions by age and gender.

Implemented time-based trend analysis using weekly and hourly impressions charts to monitor campaign performance patterns.

Built geographic analysis using a map visualization to track impressions by country.

Added interactive filters and slicers for campaign name, target interests, and dynamic metrics to enable deeper data exploration and insights.
